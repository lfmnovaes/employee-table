# Employee Table

A React application for viewing employee data with a responsive design.

## Docker Setup

### Run with Docker

```bash
docker compose up -d
```

### Access

- Frontend: http://localhost:5173
- API: http://localhost:3000/employees

### Logs

```bash
docker compose logs -f
```

### Stop

```bash
docker compose down
```

## Development Without Docker

### Prerequisites

- Node.js
- Yarn

### Setup

```bash
yarn install
yarn dev
```
